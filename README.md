## ‚öôÔ∏è Configuration

### 1. Account Setup (`accounts.json`)
Edit the `accounts.json` file to add your accounts.

```json
[
    {
        "walletAddress": "0xYourWalletAddress...",
        "sessionToken": "YourSessionTokenHere...",
        "proxy": "http://user:pass@ip:port"
    },
    {
        "walletAddress": "0xAnotherWallet...",
        "sessionToken": "AnotherToken...",
        "proxy": "http://user:pass@ip:port"
    }
]
```

> **Note:**
> - `walletAddress`: Your EVM wallet address.
> - `sessionToken`: The JWT token or session string from local storage/headers.
> - `proxy`: (Optional) Proxy URL. Leave empty/null if not using proxy.

### 2. General Config (`config.json`)
You typically don't need to touch this unless endpoints change.
