## ÔøΩ Instalasi (Cara Pasang)

Ikuti langkah-langkah ini perlahan untuk pemula:

### 1. Download Bot
Buka terminal (Command Prompt atau PowerShell) dan ketik perintah ini:

```bash
git clone https://github.com/sipaldrop/ShadeBot-Sipal.git
cd ShadeBot-Sipal
```

### 2. Install Aplikasi Pendukung
Masih di terminal, ketik perintah ini untuk menginstall semua kebutuhan bot:

```bash
npm install
```
*Tunggu sampai proses selesai. Mungkin butuh beberapa menit.*

### 3. Setup Konfigurasi
Copy file konfigurasi template:

*   Copy `config_tmp.json` dan ubah namanya menjadi `config.json`.
*   (Opsional) Buka `config.json` untuk menyesuaikan endpoint jika diperlukan.

## ‚öôÔ∏è Konfigurasi (Pengaturan)

### 1. Buat File Akun (`accounts.json`)
Kamu perlu membuat/edit file `accounts.json` untuk menyimpan data akunmu.
Isi data akunmu seperti format di bawah ini:

```json
[
    {
        "walletAddress": "0xYourWalletAddress...",
        "sessionToken": "YourSessionTokenHere...",
        "proxy": "http://user:pass@ip:port"
    },
    {
        "walletAddress": "0xAnotherWallet...",
        "sessionToken": "AnotherToken...",
        "proxy": "http://user:pass@ip:port"
    }
]
```
> **Catatan:**
> - `walletAddress`: Alamat wallet EVM kamu.
> - `sessionToken`: Token JWT atau session string dari local storage/headers.
> - `proxy`: (Opsional) URL Proxy. Biarkan kosong/null jika tidak pakai.

### 2. Pengaturan Umum (`config.json`)
File `config.json` sudah diatur otomatis. Kamu tidak perlu mengubah apa-apa kecuali ada instruksi khusus.
